### 📘 1. README.md (à la racine du projet)

# 🧩 Projet Fullstack : Application Web

Ce dépôt contient la structure de base pour une application web fullstack organisée de manière claire avec séparation des responsabilités entre le backend et le frontend.

## 📁 Structure

- `dev-backend/` : contient tout le code et la configuration du backend (API Flask).
- `dev-frontend/` : contient tout le code frontend (HTML/CSS/JS).

---

## 🚀 Objectif

Créer une architecture modulaire et maintenable pour une application web.

---

## 📌 Branches principales

- `main` : branche de référence principale.
- `dev-backend` : branche de développement dédiée au backend.
- `dev-frontend` : branche de développement dédiée au frontend.
```

# ⚙️ Backend - Flask API

Ce dossier contient le code source pour l'API backend de l'application, développée avec **Flask**.

## 🗂️ Contenu actuel

- `model.py` : définition des modèles de données.
- `config.py` : configuration de l’application.
- `route.py` : gestion des routes de l’API.
- `requirements.txt` : dépendances Python du projet.

---

## 🧪 Prochaines étapes

- Ajouter les fonctionnalités (authentification, gestion des données…).
- Créer des routes RESTful.
- Ajouter des tests unitaires.

---

## 📦 Installation

```bash

pip install -r requirements.txt
python app.py
```
```
